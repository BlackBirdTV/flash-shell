flash shell Documentation

Installation

Installing flash-shell is fairly easy. Simply follow your OS's respective guide below.

Windows

To install flash-shell on windows, you just need to download the setup from here. Next, run the setup, restart your machine and you're good to go!

Linux

Installing flash is just as easy if not easier as on windows. You'll just need to run this command:

curl "https://raw.githubusercontent.com/BlackBirdTV/flash-shell/main/setup.sh" | sh

macOS

Until I've found a maintainer for the macOS Version, I'll not release flach on macOS.

Command Line Usage

Flash is built for simplicity and ease-of-use while simultaneously being reliable and fast. A jack of all trades, if you want to say it that way. There's nothing that's not possible in flash. First of all, let's talk about the standard usecase of flash: as a command line. The next few chapters will individually explain the usage of each and every command flash has built in. However, you can also use flash like you would use cmd/bash. If flash doesn't know the command you typed in, it will automatically run it with your respective system shell.

Commands

cd

Change directory.

clear

Clear the screen

cp

Copy a file/directory from one place to another

echo

Print a file to the stdout. Use -c to not break the line afterwards.

head

Show the first ten lines of a file

info

Show info about your flash version

less

Display the contents of a file

ls

List the current directory. Optionally, give a path to a directory that you want to list.

mkdir

Create a directory

mv

Move a file/directory from one path to another

pwd

Print the working directory

rm

Remove a directory/file. Use -r to remove a non-empty directory and all its contents.

tail

Show the last 10 lines of a file

touch

Create a file

var

Declare a variable. Get its value by using $varName

Actions

Actions are like inline commands. They work by typint this: %{command name}:{input}%. E.g.: %calc:1+1%

calc

Calculate the input. Supports +, -, *, /, ^ and brackets.
